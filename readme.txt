A plugin for adding Saman Bank payment gateway to plugin edd

== Description ==

A standard plugin for adding Saman payment gateway to plugin edd which is standard itself.
<br> <br>

The capabilities of this plugin:<br>

1- 100% free.<br>

2- Payment gateway is added to edd<br>

3- RIAL is revised to ریال<br>

4- Network acceleration icon is added to edd<br>

== Installation ==

Installation: <br> <br>

First, install the plugin. Then <br> <br>

1- go to edd configuration.<br>

2- Payment gateways of the plugin.<br>

3- Saman bank gateway setting.<br>

4- Entering the received information from bank.<br>

5- Set as default payment gateway.<br>

6- enjoy it. ;)<br>
